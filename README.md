# Instructions  

1. Create a method called evenOrOdd() that takes in an integer as a parameter.</br>
Write a selection statement (if statement) that does the following:</br>

If the number is even (divisible by 2), return "Even"</br>
If the number is odd, return "Odd"</br>
If the the number is zero, return "Zero"</br>


2. Create a method called positiveOrNegative() that takes in an integer as a parameter.</br>
Write a selection statement (if statement) that does the following:</br>

If the number is positive, return "Positive"</br>
If  the number is negative, return "Negative"</br>
If the  the number is zero, return "Zero"</br>


3. Create a method called pluralize() that takes in a String as a paramter.
do the following:</br>



If the word ends in "y", return "ies"</br>
If the word ends in "ey", return "eys"</br>
If the word ends in "ife", return "ives"</br>
If none of the above are true, return "s"</br>


Accept any casing.
